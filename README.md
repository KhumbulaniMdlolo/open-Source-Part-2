# Notes writer app

Android mobile application built in Kotlin that demonstrates user authentication, settings management. The app securely handles user data with encrypted local storage and provides a simple, user-friendly interface.


Features

Create new notes instantly.
Edit existing notes.
Delete notes you no longer need.
Automatically saves your notes locally.
Simple and clean user interface.
Supports light and dark mode.

Getting Started

Android Studio (latest version)
Android SDK 24 or higher
A physical or virtual Android device
Installation Steps

Clone the repository:

https://github.com/KhumbulaniMdlolo/open-Source-Part-2.git

Open the project in Android Studio.
Allow Gradle to sync all dependencies.
Connect your Android device or start an emulator.
Run the app using:
Run > Run 'app'
